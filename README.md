设计题：
- 设计一个4轮Feistel结构的分组加密，分组的长度为8比特。半个分组为4个，即L和R长度各为4比特。
- 轮函数f=P(S ( E(R) + Ki)),  i=1,2,3,4 , 这里 + 表示异或。
- 假设R=①②③④， E(R) = ②①②③④③；
- S盒取DES的S盒，或者自己设计一个S盒，做一个表格即可，输入为6比特，输出为4比特，S(①②③④⑤⑥)=①②③④；P(①②③④)= ②④①③。
- 轮密钥编排从略，假定K1=111111，K2=111000，K3=000111，K4=000000
- 明文是自己的姓名。给出程序。给出密文。



## 算法特性
- **结构**：Feistel网络
- **轮数**：4轮加密
- **分组长度**：8比特
- **半分组**：4比特

## 轮函数
`F = P(S(E(R) ^ Ki))`

### 组件说明
- **E扩展**：4比特 → 6比特 (1234 → 212343)
- **S盒替换**：6比特 → 4比特
- **P盒置换**：4比特重排列 (1234 → 2413)

- K1 = 111111
- K2 = 111000  
- K3 = 000111
- K4 = 000000
